## 📌 Overview

This project is an **AI-powered Product Recommendation System** that allows users to describe their product requirements in natural language, such as:

> 💬 *"I want a phone under $500 with a good camera."*

The application sends the user's query to an AI model, which analyzes the available products and recommends the most suitable ones based on the user's preferences.

## 🚀 How It Works

```text
User enters product preference
            │
            ▼
React captures the query
            │
            ▼
Query sent to AI API
            │
            ▼
AI analyzes available products
            │
            ▼
Best matching products returned
            │
            ▼
Recommendations displayed to the user
```

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/your-username/AI-Product-Recommendation.git
```

Go to project folder

```bash
cd AI-Product-Recommendation
```

Install dependencies

```bash
npm install
```

Start development server

```bash
npm run dev
```

---

## 🔑 Environment Variables

Create a `.env` file in the root directory.

```env
VITE_OPENAI_API_KEY=YOUR_API_KEY
```
